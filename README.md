# catch — copie du site www.masquesdecatch.com

Copie statique du site **www.masquesdecatch.com** (boutique de masques de catch / lucha libre), réalisée le 3 juillet 2026 à la demande du propriétaire du site.

- Point d'entrée : [`index.html`](index.html)
- 720 fichiers (~37 Mo) : pages HTML, images, CSS, JavaScript
- Les liens internes ont été convertis pour fonctionner en local ou hébergés tels quels (GitHub Pages, etc.)

## Remarques

- Il s'agit d'un **instantané statique** : les fonctions dynamiques du site d'origine (panier, recherche, formulaires de contact, avis) ne fonctionnent pas hors de la plateforme Oxatis.
- Les balises de suivi statistique (StatRecorder) ont été retirées.
- Certains fichiers dynamiques (`*.asp?…`) ont été renommés (`?` → `@`) pour rester compatibles avec Windows ; les références internes ont été mises à jour.

## Utilisation

Ouvrir `index.html` dans un navigateur, ou activer GitHub Pages (Settings → Pages) pour servir le site en ligne.
